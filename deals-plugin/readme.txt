Starter architecture for a custom WordPress deals plugin.

== Description ==

This plugin provides a clean, production-ready bootstrap structure for future deals, discounts, and coupons features.

== Installation ==

1. Upload the `deals-plugin` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the "Plugins" screen in WordPress.
